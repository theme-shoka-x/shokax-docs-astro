# 列表查看

`list` 命令显示所有已管理的文章和资源，提供详细的元数据信息，便于快速了解项目的内容库存。

## 基本用法

```bash
hyc list [type]
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `[type]` | 列表类型：`posts`、`assets` 或 `all` | `all` |

## 列表类型

### posts - 仅显示文章

```bash
hyc list posts
```

显示所有 Markdown 文章及其元数据：
- 文章标题
- Slug
- 文件路径
- 发布状态
- 分类
- 创建和修改时间

### assets - 仅显示资源

```bash
hyc list assets
```

显示所有图片、媒体等资源：
- 资源路径
- 资源类型（图片、视频等）
- 文件类型（jpg、png 等）
- 是否为远程资源
- 所属文章

### all - 显示全部

```bash
hyc list
hyc list all
```

同时显示文章和资源的完整列表。

## 过滤和排序

:::tip
当前 `list` 命令显示所有内容。如果需要更多的过滤选项，可以：
1. 使用 `hyc query` 搜索特定文章
2. 结合管道命令进行文本处理（Linux/macOS）
:::

## 项目优势

<CardGrid stagger>
	<Card title="现代技术栈" icon="pencil">
		基于 **Astro + Svelte 5 + UnoCSS + Bun**，在内容站点场景下兼顾开发体验与运行性能。
	</Card>
	<Card title="优雅的 ShokaX 风格" icon="document">
		延续 ShokaX 的设计语言与阅读体验，以 Astro 生态实现独立重构，适合个人博客与内容品牌。
	</Card>
	<Card title="交互能力更强" icon="add-document">
		使用 Svelte 5 组件增强导航、搜索与动态展示能力，在静态优先架构中获得灵活交互。
	</Card>
	<Card title="主题配置清晰" icon="setting">
		通过配置文件集中管理导航、侧边栏、图标与主题行为，便于快速定制你的博客风格与信息结构。
	</Card>
	<Card title="部署与迁移友好" icon="open-book">
		内置常见部署平台配置，并提供从 Hexo 迁移的指引。可继续阅读
		[迁移说明](/guides/hexo/) 与 [术语表](/guides/glossary/)。
	</Card>
